# Avance del Proyecto

Este avance contiene los modulos trabajados por Cristofer en React + Tailwind.

Modulos incluidos:
- `Contacto`
- `ProveedoresAdmin`
- `UsuariosAdmin`

Tecnologias usadas:
- `React`
- `Tailwind CSS`
- `json-server`
- `db.json`

Rutas disponibles:
- `#/contactos`
- `#/proveedores`
- `#/usuarios`

Archivos principales:
- `codigo/ContactoPage.jsx`
- `codigo/ProveedoresAdminPage.jsx`
- `codigo/UsuariosAdminPage.jsx`
- `codigo/Header.jsx`
- `codigo/AdminShellSimple.jsx`
- `codigo/jsonApi.js`
- `codigo/appData.js`
- `db.json`

Como ejecutarlo:
1. Ejecutar `npm run api`
2. Ejecutar `npm run dev`
3. Abrir `http://localhost:5173/#/contactos`

Notas:
- `Proveedores` y `Usuarios` ya trabajan con `db.json`.
- Si `json-server` no esta encendido, la vista sigue funcionando con datos base para no romper el avance.
